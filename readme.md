# 🍷 Silent Gourmet

**취향은 공유하되 거리는 유지하는<br>내향인을 위한 데이터 기반 미식 매칭 플랫폼**

---

## 📌 Overview

**Silent Gourmet**는 혼자서는 가기 어려운 고급 미식 공간을 내향적인 사용자도 편안하게 경험할 수 있도록 설계한 AI 기반 미식 매칭 & 투어 플래너 프로젝트입니다. 

기존 소셜 다이닝 서비스의 과도한 스몰토크와 광고 중심 맛집 플랫폼의 피로감을 해결하기 위해 **"조용한 미식 경험"**에 집중했습니다.

<br>

## ✨ Core Concept

- **조용한 매칭, 깊은 미식**
- 혼밥이 어려운 오마카세 · 미쉐린 식당 문제 해결
- 대화 없이 음식만 공유하는 **Silent Mode**
- 데이터 기반 미각 취향 매칭
- 미식 투어 동선 추천 시스템
- 내향인을 위한 UX 설계

<br>

## 🧩 Problem

**1. 혼자는 예약하기 어려운 고급 식당**  
고급 다이닝은 대부분 2인 이상 예약 중심으로 운영되어 혼자 방문하기 어렵습니다.

**2. 기존 소셜 다이닝의 피로감**  
불필요한 스몰토크와 사생활 중심 대화는 내향적인 사용자에게 큰 피로로 작용합니다.

**3. 광고 중심 맛집 플랫폼**  
신뢰할 수 없는 광고형 리뷰 때문에 실패 경험이 반복됩니다.

<br>

## 💡 Solution

- **Silent Mode Matching**: 인사만 나누고 식사 중 대화를 최소화한 새로운 형태의 미식 매칭 시스템
- **Taste Radar**: 사용자의 미각 취향 데이터를 분석하여 취향 기반 미식 추천 제공
- **Gourmet Tour Planner**: 식당 → 카페 → 산책까지 하루 동선을 하나의 경험으로 설계

<br>

## 👤 Personas

### 송영욱 — 진성 내향인 미식가
> *"말 한마디보다 완벽한 소스 한 방울에 감동한다"*
- 혼자 고급 식당 가기 어려움
- 조용한 식사 환경 선호
- **Silent Mode 핵심 사용자**

### 김민석 — 효율 중심 사용자
> *"맛집 찾는 고민 자체가 귀찮다"*
- 검증된 맛집만 원함
- 실패 없는 동선 추천 선호
- **미식 투어 플래너 핵심 사용자**

### 박지윤 — 예술적 영감 수집가
> *"요리에 대한 이야기만 나누고 싶다"*
- 사적인 대화는 부담
- 음식 자체에서 영감 획득
- **미식 리포트 기능 핵심 사용자**
